## Cursor Cloud specific instructions

- The dev server is already running on [http://localhost:3001](http://localhost:3001) (launched by `.cursor/environment.json → terminals.dev`). Do not re-run `yarn start`.
- To demo UI work, open [http://localhost:3001](http://localhost:3001) in the VM browser and interact via mouse/keyboard.
- `yarn test:update` is known to be flaky in unrelated suites — don't block on it.
